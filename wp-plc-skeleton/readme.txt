A simple plugin to work with onePlace Skeleton based Modules

== Description ==

If you have created a onePlace Skeleton based Module for onePlace, and you
want to use some if its data to show on a wordpress page - this plugin is for you!

Create your own wordpress plugin in seconds - to work with your module.

* Is working with any onePlace Skeleton API compatible module

It comes with some basic shortcodes and elementor widgets

* Slider

OnePlace Connect Plugin is required !

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enable the settings you want in "WP PLC Skeleton" in the OnePlace Menu in Settings Panel
